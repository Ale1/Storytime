# storytime README

StoryTime is a VS Code extension that enables users to open their Osiris story file and story log in split screen.   
A clickable GOTO button will appear wherever a line number is mentioned in the story log. 
Clicking on the GOTO will scroll the Story window to the mentioned line. 

To start, open palette (ctrl+shift+P) and select "storytime: open story files"
